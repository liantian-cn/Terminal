# Terminal 项目总览

`Terminal` 是游戏外的 Python 程序，负责抓取 `DejaVu` 画出的矩阵，并把它还原成结构化信息。

## 它在做什么

1. 找到矩阵区域
2. 抓取矩阵画面
3. 把图像切成协议定义的像素块
4. 识别颜色、图标、文字和标记区
5. 输出结构化状态给上层使用

## 输入组成

- `Cell`: 4x4，用来表达布尔值或数值映射
- `MegaCell`: 8x8 纯色块，用来表达明显控制信息
- `BadgeCell`: 8x8 图标加脚标，用来表达技能、Buff、Debuff
- `CharCell`: 8x8 字符像素块，用来表达文本

## 关键边界

- `Terminal` 只负责“稳定还原”
- 它读取的是显示协议，不是 WoW 内部状态
- 如果协议要改，先更新本目录文档，再安排代码改动
- 不把 WoW 插件输出逻辑写进这个项目

## 当前状态

- `Terminal` 目前还是架构起步阶段
- 当前文档重点是帮助 agent 快速找到协议、分层方式和开发边界
